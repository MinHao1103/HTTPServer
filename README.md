## Jetty
```text
Jetty 輕量級的 Java Web 伺服器和 Servlet 容器
- 1. Server：能夠接收客戶端的 HTTP 請求，並向客戶端返回 HTTP 響應
- 2. Client：可以向其他 HTTP 伺服器發送 HTTP 請求，並接收並處理來自該伺服器的響應
- 3. 實現 javax.servlet API，所以可以作為 Servlet 容器，可部署基於 Servlet 的 Java Web 應用程式
- 4. 類似 Tomcat，但比 Tomcat 更輕量
```

## Jersey
```text
Jersey 是一種 Framework
- 1. 基於 JAX-RS 框架，實現 RESTful Web 服務
- 2. 類似 Spring MVC，提供 RESTful Web 服務相關的功能
```
